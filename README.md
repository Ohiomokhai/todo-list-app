# To-Do List Web Application
A responsive To-Do List application built with JavaScript, HTML, and CSS. Users can create, update, track, and delete tasks, with data persistence provided by localStorage.

## Features
- Create tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- localStorage persistence
- Responsive design
- Clean code following KISS and DRY principles

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
